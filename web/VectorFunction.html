Instructions

Enter functions of t in the input boxes, then set bounds on t and a step size to watch an animation of the vector function from minT to maxT. Regular math functions, such as sin(t) and exp(t) are allowed, as are constants like e and pi. To do e^t in the function boxes, type exp(t).
